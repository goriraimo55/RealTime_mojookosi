文字起こし ブラウザ内蔵エンジンは Chrome / Edge / Android Chrome で動作します。デスクトップ音声も文字起こししたい場合は Whisper互換API か ハイブリッド を選んでください。
Whisper互換API (音声認識) OpenAI (https://api.openai.com/v1 / whisper-1)、Groq (https://api.groq.com/openai/v1 / whisper-large-v3)、ローカルの faster-whisper-server / speaches などに対応。
